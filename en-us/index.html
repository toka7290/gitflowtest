What is TTSProjection?

TTSProjection is a projection mapping software for flat surfaces created by Toka.

Research Project in FY30

Official name: Tamura Tokai Sibuya Projection

Main Functions

  • Playback of multiple images and videos
  • Display of analog and digital clocks
  • Accurate media control by program list

Description of each mode

  • Normal mode (normal)

    • Displays images or videos as they are.

    It is also possible to display nothing for a certain period of time.

  • Clock mode (clock)

    • Displays a clock in front of the normal mode.

    Clock can be drawn programmatically without using images or videos

    The clock ticks the current time exactly!

  • Normal Plus Mode (normal+)

    • Fade in/out, rotation, and other image and video movements

  • Debug mode(debug)

    • Mode to display test patterns. For adjusting the projector.

How to use

  1. Add photos, videos, etc. todata/MediaSave photos, videos, etc. in

  2. Set the order of playback, etc. in the program list.

    By default, thedatain theprogram.xmlin data.

    For more information on how to edit the program list, click here!
  3. Start the application.

Cautions for use

If the program is saved on a removable disk, copy it to the desktop and use it.

*If you start the application from a removable disk, it may not work properly.

If the operation is not stable, it may be improved by reducing the video quality.
